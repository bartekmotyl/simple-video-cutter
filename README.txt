[English] 

* Installation

Unpack archive into a folder, for example 'simple-video-cutter'. 
Double click SimpleVideoCutter.exe to start the program.
Note: you will need ffmpeg, you can download it from https://ffmpeg.zeranoe.com/builds/

* Upgrade 

Unpack archive into an existing folder, overwriting existing content. 
File 'config.json' contains your configuration, it shouldn't be overwritten. 

This software is available for free!
Web page: https://github.com/bartekmotyl/simple-video-cutter
Email: simple.video.cutter@gmail.com

Please contact us if you would like to help improve translation to your language.
